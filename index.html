GetSUCCESS Priority Autopilot

AI Assistant for GetSUCCESS Annual Goal Generation

I have built an AI assistant to help Capgemini and Sogeti employees write their annual goals for the GetSUCCESS portal. For the tech stack, I used Streamlit for the web interface, Microsoft AutoGen to coordinate a pipeline of three specialized AI agents, and the Llama 3.3 model via Groq to power the intelligence. When employees enter their profile details (with the option to select a target grade if aiming for a promotion), specify a goal category (Assignment, Development, or Business), and add custom additional contributions (such as past achievements or future targets), these agents leverage a local knowledge base to automatically generate personalized, accurately weighted goals (ranging from 1 to 5) that can be copy-pasted directly into the HR system. Additionally, the application persists session data in a local SQLite database, allowing users to search, restore, and delete their goal-generation history.
